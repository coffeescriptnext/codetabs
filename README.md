# Code Tabs

An online editor for html, css, js with TABS

## Setup

<div class="codetabs" data-name="sketch">
   <!-- First Tab -->
   <textarea data-name="sketch.html"></textarea>
   <!-- Third Tab -->
   <textarea data-name="sketch.js"></textarea>
   <!-- Second Tab -->
   <textarea data-name="sketch.css"></textarea>
</div>
